#前期准备（软件环境配置）
#1、Nodejs
下载链接：http://nodejs.cn/download/

配置：

安装教程：https://www.runoob.com/nodejs/nodejs-install-setup.html
检验安装配置成功方法：快捷键 win + r 输入 cmd ，打开命令提示符，输入 node -v 和 npm -v 能正常显示版本号即为成功。
#2、Git
下载链接：https://git-scm.com/download/

配置：

安装教程：https://www.runoob.com/git/git-install-setup.html
检验安装配置成功方法：快捷键 win + r 输入 cmd ，打开命令提示符，输入 git –version 能正常显示版本号即为成功。（鼠标右键会有两个新的右键菜单 Git GUI Here 和 Git Bash Here )
#3、Typora（可选）(Markdown 编辑器)
下载链接：https://typora.io/#download

配置：

直接下载安装即可
(在线 Markdown 编辑器：https://pandao.github.io/editor.md/)
#安装 Hexo 框架
#1、确定文件夹路径
这个文件夹是用来放置博客的，之后的命令就在这个文件夹运行。
这里以 D:\web\hexo 路径为例。
#2、安装 Hexo 框架
npm install -g hexo-cli //安装hexo的基础框架
hexo init //初始化hexo框架
npm install //安装所需要的组件
hexo g //编译生成静态页面 (hexo generate)
hexo s //启动本地服务 (hexo server)
#3、本地 web 搭建完成
在输入 hexo s 后会出现以下代码

INFO Validating config
INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.
此时浏览器输入 <http://127.0.0.1:4000 > 即可打开网页

默认首页页面

本地服务器关闭： Ctrl + C ，然后按照提示输入 y 即可。
